## Basics

REPL (Read Evaluate Print Loop):

```sh
scala
```

or SBT REPL (same repl):

```sh
sbt
console

> def radius = 10
> 34+65
```

## Definitions

```scala
def radius = 10
def square(x: Double) = x * x
```
